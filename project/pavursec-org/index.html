pavursec.org

In addition to serving as the standard bootstrap-heavy web portfolio. This site provides a location for me to consolidate my thoughts, projects, and web persona as well as an opportunity to experiment with new technologies.

As I work on new projects they’ll be added to this site - wherever they may be hosted. It also hosts my blog where I’ll share my thoughts on cyberpolicy, security research discoveries and disclosures, and general technical musings.

The site itself is built using the static template engine “Hugo” and a custom bootstrap theme I applied my limited design skills towards assembling. At some point I may release the template publicly (you’ll see it on my projects page when I do). By keeping the site static I hope it remains scalable and secure.
